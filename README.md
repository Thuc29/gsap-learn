## Spylt Portfolio

GSAP-powered portfolio built with Next.js App Router.

### Tech
- Next.js, React, TypeScript
- GSAP for motion
- Tailwind CSS

### Development
```bash
npm install
npm run dev
```
Visit http://localhost:3000.

### Scripts
- `npm run dev` – start the dev server
- `npm run lint` – lint the project

### Notes
- Main layout: `app/layout.tsx`
- Entry page and sections: `app/page.tsx` with components in `src/compoments`
